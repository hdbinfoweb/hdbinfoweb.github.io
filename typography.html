Typography

In addition to establishing a consistent look and feel throughout the website, a quality typeface ensures strong legibility and readability. Cabin is the font of choice for the HDB InfoWEB. A humanist font with strong roots in modernism, it is clean and contemporary whilst remaining personable and friendly. It can be used for both headlines and content.

With regards to technical implementation, we have a selection of fall-back fonts that will be called in case there is any error in loading the Cabin font. In the CSS, the full sequence is

font-family: ‘cabin’, Helvetica, Arial, sans-serif

At each step, the browser will check if the font is available locally and if all else fails, fall back on the browser-specified default for a sans serif typeface.

  • Cabin

    Cabin Regular is used across all content. Cabin Bold should only be used for titles and words you want to emphasise.

  • Helvetica

    Helvetica is the secondary font choice. As it has a similar width and height to Cabin, it can help you avoid any major restructuring of the content.



Font sizes. By using a standardised set of weights and sizes across copy, we establish a visual hierarchy which makes it easier for the user to scan the page and increases overall accessibility.



Application. Clear guidance on how the typography is being applied will ensure consistency throughout your site.
