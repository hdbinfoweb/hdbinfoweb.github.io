Buttons

Website elements help in delivering information in a clear and concise way.



Throughout the HDB InfoWEB, a visual language is used that aims to be as consistent as possible to aid user experience and expectation.

As the colour palette within the HDB Brand consists of a core set of colours (namely the HDB Red and her shades, black and grey tones, as well as a teal that has been made Web Interface Standard compliant (WIS) - we have very deliberately applied these colours across the board in a consistent manner:

  • Primary, HDB Red

    Denotes key calls-to-action that bring the user to a new page.

  • Secondary, Teal

    Denotes calls-to-action that initiate action on the page itself.

  • Coloured background, White

    Denotes calls-to-action that are used on dark backgrounds.

  • Light Grey

    Used for chevrons on callsto- action.



Clear and easy to use interaction is important to the overall user experience of a site. By ensuring that active elements (such as links and buttons) provide consistent visual feedback on interaction, we aid ease of navigation and overall accessibility.

Calls-to-action should be clearly signposted and stay consistent across all digital platforms.

  • Links should never say ‘click here’. That transfers a user’s attention away from the website interface and onto the mouse mechanics instead.
  • Be specific with your link. The word ‘here’ does not describe what will be viewed upon clicking. Instead, find the best words for the link – these are usually the nouns within the sentence.
